# Vision-Based Adaptive Traffic Signal Control System

## Project Overview
This project uses Computer Vision and YOLO-based vehicle detection to dynamically control traffic signals based on real-time traffic density.

## Technologies Used
- Python
- OpenCV
- YOLO
- ESP32
- Adafruit IO

## Features
- Real-time vehicle detection
- Lane-wise traffic density analysis
- Adaptive signal timing
- IoT-based monitoring using ESP32

## Workflow
1. Capture traffic video.
2. Detect vehicles using YOLO.
3. Calculate vehicle density.
4. Adjust signal timing automatically.
5. Monitor data through Adafruit IO.

## Outcomes
- Reduced traffic congestion.
- Improved traffic flow efficiency.
- Automated traffic signal management.
